## Documentation

### Installation

Install Ruby and Bundler at the user level.

Clone the project or your copy of the template.

Install project dependencies:

```sh
$ make install
```

### Usage

Start dev server:

```sh
$ make serve
```

Open in the browser:

- http://localhost:4000

Build:

```sh
$ make build
```
